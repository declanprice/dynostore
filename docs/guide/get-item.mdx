# GetItem

### Get item

```tsx
import { DynamoDBClient } from '@aws-cdk/client-dynamodb'
import { Store } from '@declanprice/dynostore'

const store = new Store('CustomerTable', new DynamoDBClient())

const customer = await store.get<CustomerItem>().exec();
```

### Get item with strong consistency

```tsx
const customer = await store.get<CustomerItem>().consistent().exec();
```


### Get item with projection

```tsx
const customer = await store.get<CustomerItem>().project('id,fistName,lastName').exec();
```

### Get item for transaction

Get item can be used within a transaction

```tsx
const customerTxItem = store.get<CustomerItem>().tx();
```
