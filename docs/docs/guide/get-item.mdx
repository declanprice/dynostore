# GetItem

### Store.get()

```tsx
import { Store } from '@declanprice/noxa'

const store = new Store('customers', new new DynamoDBClient())

const getItem = await store.get<CustomerItem>();
```

### key(key)

Get item must always contain a key object.

```tsx
getItem.key({ id: customer.id });
```

### consistent(key)

Get item can optionally be read the item with strong consistency.

```tsx
getItem.consistent();
```

### project(projection)

Get item can optionally project specific attributes using comma seperated values.

```tsx
getItem.project('id,fistName,lastName')
```


### exec()

Get item will only be run once you specifically execute it.

The response from exec will be the unmarshalled item.

```tsx
const customer: Customer = getItem.exec();
```

### tx()

Get item can be used within a transaction

```tsx
const getCustomerTx = getItem.tx();

await transactGetItems(getCustomerTx);
```
